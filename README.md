# Entropy Breakers — Landing Page

Production-ready landing page for Bettina Baranyi's Angol Mentor Program.

## Structure
```
landing-site/
  index.html          (49 KB — HTML + CSS + JS)
  images/             (1.65 MB total, 14 files)
    bettina-*.png     (3 portraits, transparent PNG)
```

## Deploy
- **GitHub Pages**: push to `main`, enable Pages in settings → it's live
- **Cloudflare Pages**: drag and drop the folder, done
- **Netlify**: same — drag and drop

## Edit
- Copy: edit `index.html` directly (look for Hungarian text)
- Images: replace files in `images/` folder (keep same filenames)
- Prices: search for `35 000 Ft` / `65 000 Ft` / `95 000 Ft` in index.html
